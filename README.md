# Nooro Todo Frontend (Next.js App Router + Tailwind, TypeScript)

## Setup
```bash
npm install
echo "NEXT_PUBLIC_API_URL=http://localhost:4000" > .env.local
npm run dev
```
Open http://localhost:3000

## Pages
- `/` Home: list, toggle, delete, navigate to edit
- `/tasks/new`: create
- `/tasks/[id]/edit`: edit
